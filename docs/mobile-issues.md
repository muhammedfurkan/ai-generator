# Cilt İyileştirme Mobil Sorunları

Videodan tespit edilen sorunlar:
1. Sayfa üstünde header yok - sadece boş alan var
2. Scroll yapıldığında sayfa çok uzun ve bottom nav ile çakışıyor
3. Sidebar mobilde görünmemeli - tek sütun layout olmalı
4. Fiyatlar: 3 kredi → minimum 20 kredi olmalı

Düzeltilecekler:
- Mobil header ekle
- Mobil için tek sütun layout
- Sidebar bilgilerini mobilde aşağıya taşı veya gizle
- Fiyatları güncelle (20, 25, 30 kredi)
